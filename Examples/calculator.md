# calculator function
#### This is an example of calculator
<br>

## There is no customization
```js
const simplydjs = require('simply-djs')

// messageCreate event
// calculator command
simplydjs.calculator(message, {
    embedColor: 'hex code', //default: #075FFF
})
```
